# ويبي نت | WeebyNet Store 🛒

موقع إلكتروني احترافي لشركة ويبي نت للأجهزة الإلكترونية.

## 🔗 رابط الموقع
بعد رفعه على GitHub Pages يصير الرابط:
```
https://<username>.github.io/<repo-name>/
```

## 📦 محتوى الموقع
- 🛍️ سلة تسوق تفاعلية
- 💳 نظام دفع متعدد الخطوات
- 📱 متوافق مع الموبايل
- 🔍 بحث وفلترة المنتجات
- 6 فئات: هواتف، حواسيب، لوحية، كهربائي، تعليمي، إكسسوارات

## 🚀 طريقة الرفع على GitHub Pages
1. ارفع الملفات على GitHub
2. روح **Settings → Pages**
3. اختار **Branch: main** ثم **/root**
4. اضغط Save وانتظر دقيقة

## 🛠️ التقنيات المستخدمة
- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Cairo, Tajawal)
- تصميم RTL عربي كامل
